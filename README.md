# 🇺🇸 Survival Dream — 异乡求生记

> *在这颗星球上最富裕的国度里，穷是一种传染病。而你，已经确诊了。*

一款以**文字叙事 + 资源管理**为核心的回合制生存策略游戏。你将扮演一名漂洋过海的异乡人，在底层社会中挣扎求活，体验从「一无所有」到「阶层跃迁」的全过程。

---

## 🎮 怎么玩

### 游戏目标

**活下去，并且尽可能活得更好。**

游戏没有固定的通关条件——你可以苟在温饱线上勉强度日，也可以搏一把冲击财务自由的巅峰（竞选地方议员、创办科技公司、入手超跑……）。但更大的概率是——你会倒下。

### 游戏流程

```
选择难度 → 选择身份 → 观看入境剧情 → 开局任务（第一周）→ 自由生存回合
```

游戏分为 **6 个阶段**：

| 阶段 | 名称 | 说明 |
|------|------|------|
| S00 | 窒息的日常 | 开场剧情：内卷与迷茫 |
| S01 | 序章 | 底层数据揭示，铺垫基调 |
| S02 | 身份档案 | **选择你的身份**（核心分歧点） |
| S03 | 入境 | 根据身份展开不同的入境故事线 |
| S04 | 一切清零 | 过渡剧情，进入生存模式 |
| S05 | 生存速成 | 第一周的开局任务，学会基本操作 |
| 🔁 | 自由生存 | 无限回合制，每回合选择行动直到倒下或发达 |

游戏开局需要选择一个**身份档案**，不同身份决定了：
- 🎬 完全不同的入境剧情

- 🏠 不同的第一周开局任务和起始住所

| 身份 | 初始资金 | 初始体力 | 初始精神 | 初始评分 | 特点 |
|------|---------|---------|---------|---------|------|
| 🎓 留学生（学生签证） | $10,000 | 70 | 55 | 580 | 合法身份但随时过期，工作签证像抽奖 |
| 🏃 丛林穿越者（徒步偷渡） | $10,000 | 40 | 40 | 560 | 徒步穿越热带雨林，最艰苦的开局 |
| 📦 铁箱客（海运偷渡） | $10,000 | 50 | 45 | 560 | 铁皮箱中漂流三周，精神严重受创 |

> 💡 **丛林穿越路线**体力和精神都很低，属于地狱难度开局；**留学生路线**相对均衡，适合新手。

## 📊 四大核心属性

每个属性都是你的**生命线**，任何一项清零都可能导致游戏结束。

### 💰 资金（Money）
- 你的命根子。在底层社会，连呼吸都在烧钱
- 每回合自动扣除：**租金** + **伙食费**
- 归零后无法执行大部分行动，持续负债会导致倒下

### ❤️ 体力（Health）
- 你的身体状态，范围 0~100
- 透支劳动、受伤、生病都会消耗体力
- **清零 = 躯体崩解 = 游戏结束**

### 🧠 精神值（SAN）
- 你的心理承受能力
- **每个行动都消耗精神值**（这是最重要的限制资源！）
- 精神值上限由**住所等级**决定（露宿 100 → 海滨豪宅 250）
- **清零 = 意识坍塌 = 游戏结束**

### 📈 信用评分（Credit）
- 社会的隐形通行证
- 影响能否申办信用卡、借款、保险等
- 每回合自然衰减 -10，需要主动维护
- 部分高级行动需要评分达标才能解锁

## 🎯 行动系统

每个回合你可以执行多个行动，但受限于**精神值**。行动分为 **8 大类**：

| 类别 | 图标 | 说明 | 典型行动 |
|------|------|------|---------|
| 治疗 | 🏥 | 恢复体力 | 超市折扣药、跨境买药、急诊白蹭、网络众筹求助 |
| 打工 | 🧱 | 赚钱（核心收入） | 路边举牌、跑网约车、跑腿送餐、出售血浆、物流仓库搬运 |
| 拼命 | 💀 | 以体力换资金 | 出售器官、密集卖血、高危工种 |
| 花钱续命 | 💎 | 以资金换体力 | 高端私立医院、健身中心、心理疏导、购买医保 |
| 信贷 | 💳 | 债务工具 | 短期高息贷、申办信用卡、准时还款、申请个人破产 |
| 豪赌 | 🎰 | 高风险高回报 | 买刮刮乐、炒迷因股票、全仓虚拟币、地下牌局 |
| 喘息 | 🛋️ | 恢复精神 / 放空 | 桥洞躺尸、教堂施粥、看场电影、主题乐园一日游 |
| 跃升 | 👑 | 阶层跳板 | 种子轮投资、开连锁食店、入手超跑、创办科技公司、竞选地方议员 |

### 行动类型

每个行动都有不同的**结果模式**：

| 类型 | 说明 | 示例 |
|------|------|------|
| ✅ 确定型 | 结果固定，无随机性 | 超市折扣药：-15 SAN，+17 体力 |
| 🎲 概率型 | 多个结果按概率触发 | 跑网约车：30%好评大赏 / 40%普通 / 30%被投诉 |
| ⚠️ 冒险型 | 有稳定收益，但可能触发负面效果 | 物流仓库搬运：赚$5500，但25%概率闪腰 |
| 🎰 博彩型 | 纯看运气，可能血本无归 | 买刮刮乐：99%白搭，1%中$50,000 |

### 解锁条件

并非所有行动一开始就可用：

- **默认解锁**：路边举牌、超市折扣药、买刮刮乐等
- **回合解锁**：跑腿送餐（第3回合）、全仓虚拟币（第7回合）等
- **条件解锁**：捡瓶子换钱（资金<500时）、大厂资深码农（资金>30000时）、创办科技公司（资金>120000时）

## 🧠 新手攻略

### 生存优先级
1. **守住精神值** — 精神是一切行动的根基。没有精神值，你什么都做不了
2. **确保进账 > 出账** — 每回合租金+伙食是固定开销，必须覆盖
3. **维护信用评分** — 评分每回合自然 -10，定期「准时还款」维护
4. **量力升级住所** — 住所提升精神上限，但租金也会暴增

### 常见翻车原因
- 😵 精神值耗尽：行动太激进，没留精神值做恢复
- 💸 入不敷出：升了市中心公寓但没有稳定高收入
- 🤒 体力死亡螺旋：受伤→没钱治→继续干活→伤上加伤
- 🎰 赌博清零：全仓虚拟币，一觉醒来归零了

### 翻身路线
1. **稳扎稳打**：打工 → 攒钱 → 升住所 → 提升精神上限 → 解锁高收入行动
2. **豪赌路线**：攒够本金 → 炒迷因股/虚拟币 → 运气好直接起飞（运气不好直接归零）
3. **跃升路线**：积累资金到 $50K+ → 种子轮投资/开连锁食店 → 创办科技公司 → 竞选地方议员
