# 🇺🇸 Survival Dream — 美利坚生存指南

> *你以为最大的冒险是年底考核。命运在一夜之间把你推进了逃亡者的队列。*

一款以**文字叙事 + 资源管理**为核心的回合制生存策略游戏。你将扮演一名被卷入腐败案件的前公务员，仓皇出逃至大洋彼岸，在底层社会中挣扎求活，体验从「一夜归零」到「重建人生」的全过程。

---

## 🎮 怎么玩

### 游戏目标

**活下去，并且尽可能活得更好。**

游戏没有固定的通关条件——你可以蜷缩在温饱线上苟一天算一天，也可以拼命往上爬冲击人生巅峰（创办科技公司、开中餐连锁、参选市议员……）。但更大的概率是——你会倒下。

### 游戏流程

```
选择难度 → 选择入境方式 → 观看入境剧情 → 开局任务（第一周）→ 自由生存回合
```

游戏分为 **6 个阶段**：

| 阶段 | 名称 | 说明 |
|------|------|------|
| S00 | 一纸通告 | 开场剧情：上司落马，你被当成挡箭牌 |
| S01 | 序章 | 底层数据揭示，铺垫基调 |
| S02 | 落脚方式 | **选择你的入境方式**（核心分歧点） |
| S03 | 入境 | 根据路线展开不同的入境故事 |
| S04 | 一切归零 | 过渡剧情，进入生存模式 |
| S05 | 生存速成 | 第一周开局任务，学习基本操作 |
| 🔁 | 自由生存 | 无限回合制，每回合选择行动直到倒下或翻身 |

- 🎬 完全不同的入境故事

- 🏠 不同的第一周开局任务和起始住处

| 路线 | 初始资金 | 初始体力 | 初始精神 | 初始评分 | 特点 |
|------|---------|---------|---------|---------|------|
| 🎓 访问学者（公派签证） | $12,000 | 75 | 60 | 590 | 合法身份但三个月到期，时间不站在你这边 |
| ✈️ 旅游签滞留者（逾期不归） | $8,000 | 65 | 50 | 560 | 撕掉回程票成了幽灵，靠华人社区谋生 |

> 💡 **蛇头路线**体力和精神都极低，属于地狱难度开局；**访问学者路线**最均衡，适合新手。

## 📊 四大核心属性

每个属性都是你的**生命线**，任何一项清零都可能导致游戏结束。

### 💰 资金（Money）
- 你的命根子。在底层社会，连喘气都在烧钱
- 每回合自动扣除：**租金** + **伙食费**
- 归零后无法执行大部分行动

### ❤️ 体力（Health）
- 你的身体状态，范围 0~100
- 过劳、受伤、营养不良都会消耗体力
- **清零 = 身体报废 = 游戏结束**

### 🧠 精神值（SAN）
- 你的心理承受极限
- **每个行动都消耗精神值**（最重要的限制资源！）
- 精神值上限由**住所等级**决定（睡大街 100 → 海景豪宅 250）
- **清零 = 精神瓦解 = 游戏结束**

### 📈 信用评分（Credit）
- 美国社会的隐形通行证
- 影响能否办信用卡、贷款、保险等
- 每回合自然衰减 -10，需要主动维护
- 部分高级行动需要评分达标才能解锁

## 🎯 行动系统

每个回合你可以执行多个行动，但受限于**精神值**。行动分为 **8 大类**：

| 类别 | 图标 | 说明 | 典型行动 |
|------|------|------|---------|
| 医疗 | 🏥 | 恢复体力 | 自己买药硬扛、过境去墨西哥抓药、混进急诊赖账、GoFundMe求救 |
| 搞钱 | 🔨 | 赚钱（核心收入） | 街头卖艺、开网约车、中餐馆后厨、卖血浆、亚马逊仓库分拣 |
| 透支 | 💀 | 拿健康换现金 | 黑市卖肾、高频卖血、有毒环境工作 |
| 花钱保命 | 💎 | 拿钱换健康 | 私立诊所VIP、健身房年卡、看心理医生、买医疗保险 |
| 信用 | 💳 | 债务工具 | 发薪日贷款、办预付信用卡、按时还账、申请个人破产 |
| 赌 | 🎰 | 高风险高回报 | 买彩票、炒Meme股、All in加密货币、华人地下赌场 |
| 休整 | 🛋️ | 恢复精神 / 放空 | 找个角落瘫着、教堂免费午餐、一个人去看电影、迪士尼一日游 |
| 翻盘 | 👑 | 阶层跃迁 | 天使投资、开中餐连锁、买野马跑车、创办科技公司、参选市议员 |

### 行动类型

每个行动都有不同的**结果模式**：

| 类型 | 说明 | 示例 |
|------|------|------|
| ✅ 稳定型 | 结果固定，不靠运气 | 自己买药硬扛：-15 SAN，+17 体力 |
| 🎲 随机型 | 多个结果按概率触发 | 开网约车：30%大额小费 / 40%一般 / 30%被投诉 |
| ⚠️ 冒险型 | 有基础收益，但可能翻车 | 亚马逊仓库分拣：赚$5500，但25%概率闪腰 |
| 🎰 赌博型 | 纯看运气 | 买彩票：99%白搭，1%中$50,000 |

### 解锁条件

并非所有行动一开始就可用：

- **默认解锁**：街头卖艺、自己买药、买彩票等
- **回合解锁**：中餐馆后厨（第3回合）、All in加密货币（第7回合）等
- **条件解锁**：翻垃圾捡废品（资金<500时）、硅谷码农（资金>30000时）、创办科技公司（资金>120000时）

## 🧠 新手攻略

### 生存优先级
1. **守住精神值** — 精神是一切行动的根基。没有精神值，你什么都做不了
2. **确保进账 > 出账** — 每回合租金+伙食是固定开销，必须覆盖
3. **维护信用评分** — 评分每回合自然 -10，定期「按时还账」维护
4. **量力升级住所** — 住所提升精神上限，但租金也会暴增

### 常见翻车原因
- 😵 精神值耗尽：行动太激进，没留精神值做恢复
- 💸 入不敷出：升了正经公寓但没有稳定高收入
- 🤒 体力死亡螺旋：受伤→没钱治→继续干活→伤上加伤
- 🎰 赌博清零：All in 加密货币，一觉醒来交易所跑路了

### 翻身路线
1. **稳扎稳打**：打工 → 攒钱 → 升住所 → 提升精神上限 → 解锁高收入行动
2. **赌狗路线**：攒够本金 → 炒Meme股/加密货币 → 运气好直接起飞（运气不好直接归零）
3. **翻盘路线**：积累资金到 $50K+ → 天使投资/开中餐连锁 → 创办科技公司 → 参选市议员
